# 🛍️ Amazon Clone (HTML & CSS)

A simple front-end clone of Amazon’s homepage created using only **HTML and CSS**.  
This project focuses on layout design, structure, and responsive styling to practice front-end development skills.

## 💡 Features
- Static Amazon-like homepage
- Navigation bar with logo, search bar, and cart icon
- Product display sections
- Footer with basic links
- Responsive design using Flexbox and Grid

## 🛠️ Technologies Used
- HTML5  
- CSS3
